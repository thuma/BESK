BESK

Anmälan och hantering av närvaro.

  1. Skapa en kodatuga under kostugor.
  2. Föräldrar anmäler sina barn på: https://besk.kodcentrum.se/apply
  3. Markera de som ska bjudas in som deltagare och tryck på bjud in så kommer de att få ett mail.
  4. När föräldrarna har svarat genom att trycka på länken i mailet så sparas det i systemet.

Textformat

När du fyller i texten som ska vara till E-Post meddelande i systemet så används Markdown som är ett enkelt format som används för att skapa texter med tillexempel länkar och rubriker.

Om du önskar ytteligare egenskaper på ditt meddelande så kan du lägga in HTML-kod också. För att tillexempel ändra storlek på texten eller teckensnitt.

Variabler i texter

Det finns variabler som kan ingå i texterna för att individanpassa meddelanden.
%namn% = Deltagarens förnamn
%kodstuga% = Kodstugans namn
%länk% = Länk för att tillexempel bekräfta plats.
%datum% = Som gäller för kodstugan.
